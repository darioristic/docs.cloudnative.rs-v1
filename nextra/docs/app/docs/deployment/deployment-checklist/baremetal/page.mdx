# Bare Metal Deployment Checklist

This checklist provides a comprehensive guide for deploying OpenShift clusters on bare metal infrastructure, ensuring optimal performance and resource utilization.

## Pre-Installation Planning

### Hardware Requirements
- [ ] **Server Specifications**
  - [ ] Minimum 3 control plane nodes
  - [ ] Minimum 3 worker nodes
  - [ ] 16+ CPU cores per node
  - [ ] 64+ GB RAM per node
  - [ ] 100+ GB storage per node
  - [ ] Network interface cards (NICs)

- [ ] **Storage Requirements**
  - [ ] Local storage configured
  - [ ] Storage performance validated
  - [ ] Backup storage available
  - [ ] Storage redundancy configured

- [ ] **Network Requirements**
  - [ ] Network switches configured
  - [ ] Load balancer hardware
  - [ ] Firewall configuration
  - [ ] Network segmentation

### Infrastructure Setup
- [ ] **Physical Environment**
  - [ ] Rack space allocated
  - [ ] Power requirements met
  - [ ] Cooling systems operational
  - [ ] Network cabling completed

- [ ] **Network Configuration**
  - [ ] IP address ranges allocated
  - [ ] DNS servers configured
  - [ ] NTP servers configured
  - [ ] Network policies defined

## Installation Process

### 1. Infrastructure Preparation
- [ ] **Hardware Setup**
  - [ ] Servers installed in racks

- [ ] **Network Configuration**
  - [ ] Switch configuration

### 2. Operating System Installation
- [ ] **OS Preparation**
  - [ ] RHEL/CentOS installed

- [ ] **Network Configuration**
  - [ ] Static IP configuration
  - [ ] DNS resolution setup
  - [ ] NTP synchronization

### 3. OpenShift Installation

  - [ ] Bootstrap node preparation

  - [ ] Bootstrap process started

- [ ] **Control Plane Installation**

  - [ ] Control plane services installed

  - [ ] Control plane configuration

- [ ] **Worker Node Installation**
  - [ ] Worker nodes prepared
  - [ ] Worker services installed
  - [ ] Worker node health verified
  - [ ] Worker node configuration
